# BuildX Analytics Project — Neha Anna Thomas

## Dataset
**Theme:** Education
**Dataset used:** College Placement Records
**Source:**  https://www.kaggle.com/datasets/benroshan/factors-affecting-campus-placement
**Reason for choosing:** I selected this dataset because it provides information about student's academic performance, work experience, placement status, specialization, and salary. It helps analyze the factors that influence campus placements.

## Questions I Answered
1. What is the average salary for each specialisation? (specialisation, salary)
2. How many students were placed and not placed? (status)
3. Which students scored between 70 and 90 in SSC percentage? (ssc_percentage)

## Key Insight
Out of 215 students, 148 were placed, giving a placement rate of 68.84%.

## Tools Used
- **Python (Google Colab):** Cleaned the dataset by handling missing values, removing duplicates, cleaning text columns, correcting data types, creating a new column, and saving the cleaned CSV.
- **SQL (SQLiteOnline.com):** Wrote SQL queries using WHERE, GROUP BY, ORDER BY, HAVING, and BETWEEN to analyze the data.
- **Power BI:** Built an interactive dashboard with multiple visuals, a DAX calculated column, a DAX measure, a slicer, a page-level filter, and a Key Insights section.
